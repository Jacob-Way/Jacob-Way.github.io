N-fold Mixing

Dynamical System Formulation

Dynamical System: A dynamical system is a 4-tuple (X,\mathcal B,\mu,f), where (X,\mathcal B,\mu) is a measure space and f:X\to X is a function modeling some dynamics on the space X

Measure Preserving: A dynamical system is measure-preserving if \mu(f^{-1}(A)) = \mu(A) for all A\in \mathcal B.

Mixing: A dynamical system is mixing if \lim_{n\to\infty}\mu(f^{-n}(A)\cap B) =\mu(A)\mu(B) for all A,B\in \mathcal B

Note: When compared to higher order mixing, this is called 2-fold mixing, since it involves 2 sets.

Example: An easy way to understand this equation is using the example of mixing paint. If X represents the inside of a can of white paint, \mu measures the relative volume of regions in the bucket, and f models the effect of stirring the bucket, we can imagine replacing region B with blue paint, then stirring n times, then scooping paint from region A. If our stirring function is mixing, we can expect the proportion of the paint in region A which is blue (or which originated in region B) to be \mu(B), the same as the proportion of blue paint before we stirred the bucket. Also, this fact will be independent of our choice of A and B.

3-fold mixing: A dynamical system is 3-fold mixing if \lim_{n\to\infty, m\to\infty}\mu(f^{-m-n}(A)\cap f^{-n}(B)\cap C) =\mu(A)\mu(B)\mu(C) for all A,B,C\in \mathcal B

Note: Using the paint mixing example, if f is 3-fold mixing, we could instead use two buckets of white paint, initially put blue paint in region C, stir m times, then scoop region B out and place it it region B of a second bucket of white paint, then mix n more times and scoop out region A. The proportion of blue paint in this scoop should approach \mu(C)\mu(B) if we mix more.

Note: We can easily see that 3-fold mixing implies 2-fold mixing by letting A=X. However, whether 2-fold mixing implies 3-fold mixing is surprisingly unknown.

Example: A common example of a dynamical system is a shift map. In this system, the space is a sequence space, and the transformation function shifts a sequence one term to the left or right. For example, X = \{0,1\}^{\mathbb Z}, f(x)_i := x_{i+1}, with \mu being the coin-toss measure. This system is mixing since different elements in any sequence are independently chosen, so any "cylinder set", which is a set defined by restricting a finite number of elements of the sequence, becomes independent from its translation eventually. By Dynkin's \pi-\lambda theorem, we only need to consider cylinder sets.
We can use this space to exhibit two near-counterexamples which are 2-fold mixing but not 3-fold mixing.

General Formulation

Using the example of the shift space, we can define mixing in a similar way for other translation spaces, which may have several ways of translating, instead of just one. We will find that, in this generalization, 2-fold mixing does not imply 3-fold mixing.

Definition: Consider a subspace of X = \{0,1\}^{\mathbb Z^2}, with a family of shift functions which shift a point in this space an integer amount in each direction. We will call a system mixing if \mu((A-nl)\cap B) \to \mu(A)\mu(B) as n\to \infty for all l\in \mathbb Z^2, 3-fold mixing if \mu((A-nl)\cap(B-mp)\cap C) \to \mu(A)\mu(B)\mu(C) as n,m\to \infty for all l,p\in \mathbb Z^2.

Example: We can show that the entire space X, with the coin toss measure and the family of translations forms a mixing and n-fold mixing system. The reason is that for every cylinder set, depending on a finite number of terms in a lattice, we can enclose the terms determining the set by a finite frame, and for a set of n cylinder sets, we can choose a finite frame that determines all of the sets. since every set of different terms in the lattice is independently chosen, and any shift applied over and over will eventually lead to a frame disjoint from its original position, if \{X_n\} is a collection of cylinder sets depending on a finite frame, for any collection of shifts \{l_n\}, \{(X_n+l_nm_n)\} is independent for some choice of \{m_n\}. Therefore, the system is N-fold mixing for all N.

Counterexample: Using these new definitions for mixing, we can show that 2-fold mixing does not imply 3-fold mixing using a similar example. Define Y = \{x\in X:x_{i,j}+x_{i+1,j}+x_{i,j+1} = 0\mod 2\}. Notice that, using this relationship, our choice of x_{i,j} for \{(i,0):i\in \mathbb Z\}, \{(j,0):j\leq -1\} allows us to determine x_{i,j} for all other (i,j). Therefore, a natural measure to use for Y is a coin toss measure defined using this subset of terms. Also notice that, because of the way that lattices in Y can be generated,
the terms with coordinates in R_1 = \{(i,j):i,j\geq 0\} are completely determined by those with coordinates in S_1 = \{(i,0):i\geq 0\},
the terms with coordinates in R_2 = \{(i,j):i\geq 0, j\leq-i-1\} are completely determined by those with coordinates in S_2 = \{(0,j):j\leq -1\},
the terms with coordinates in R_3 = \{(i,j):i\leq -1,j\leq -i-1\} are completely determined by those with coordinates in S_3 = \{(i,0):i\leq -1\}
Finally, we can show that \mu is translation invariant [###]. Using this fact, given any frame and any translation vector, it is possible to translate the frame a finite number of times by that vector and then translate the original frame and the new frame so that they lie in separate regions R_i [###]. This shows that the system we have defined is 2-fold mixing.
Now, we can add the relationship from before, with the relationship of two adjacent terms to get: x_{i,j}+x_{i+1,j}+x_{i,j+1}+x_{i+1,j}+x_{i+2,j}+x_{i+1,j+1}+x_{i,j+1}+x_{i+1,j+1}+x_{i,j+2}= 0\mod 2 \implies x_{i,j}+x_{i+2,j}+x_{i,j+2}= 0\mod 2. By induction, we also have x_{i,j}+x_{i+2^n,j}+x_{i,j+2^n}= 0\mod 2 for all n. This shows, using the cylinder sets A = B = C = \{x\in X:x_{0,0}=1\} with \mu(A) = 1/2, and the translation vectors l = (1,0), p = (0,1), That \mu((A-2^nl)\cap(A-2^np)\cap A) = 0 \neq \mu(A)^3=1/8 (since 1+1+1 \not = 0 \mod 2) for all n, so the system is not 3-fold mixing.

Note: The reason that this is not a counter-example for the 2-fold mixing implies 3-fold mixing in the dynamical system formulation is because in the above shift space, we have an entire of translation functions, while in a dynamical system, we only have one. It is impossible to find 3 frames in the above example which depend on eachother but are arbitrarily far apart in a single direction. However, there is an example of a 1-d shift space which is not 3-fold mixing but is 2-fold mixing conditional to the sigma-algebra generated by a factor.
