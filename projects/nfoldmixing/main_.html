N-fold Mixing

Dynamical System: A dynamical system is a 4-tuple (X,\mathcal B,\mu,f), where (X,\mathcal B,\mu) is a measure space and f:X\to X is a function modeling some dynamics on the space X

Measure Preserving: A dynamical system is measure-preserving if \mu(f^{-1}(A)) = \mu(A) for all A\in \mathcal B.

Mixing: A dynamical system is mixing if \lim_{n\to\infty}\mu(f^{-n}(A)\cap B) =\mu(A)\mu(B) for all A,B\in \mathcal B

Note: When compared to higher order mixing, this is called 2-fold mixing, since it involves 2 sets.

Example: An easy way to understand this equation is using the example of mixing paint. If X represents the inside of a can of white paint, \mu measures the relative volume of regions in the bucket, and f models the effect of stirring the bucket, we can imagine replacing region B with blue paint, then stirring n times, then scooping paint from region A. If our stirring function is mixing, we can expect the proportion of the paint in region A which is blue (or which originated in region B) to be \mu(B), the same as the proportion of blue paint before we stirred the bucket. Also, this fact will be independent of our choice of A and B.

3-fold mixing: A dynamical system is 3-fold mixing if \lim_{n\to\infty, m\to\infty}\mu(f^{-m-n}(A)\cap f^{-n}(B)\cap C) =\mu(A)\mu(B)\mu(C) for all A,B,C\in \mathcal B

Note: Using the paint mixing example, if f is 3-fold mixing, we could instead use two buckets of white paint, initially put blue paint in region C, stir m times, then scoop region B out and place it it region B of a second bucket of white paint, then mix n more times and scoop out region A. The proportion of blue paint in this scoop should approach \mu(C)\mu(B) if we mix more.

Note: We can easily see that 3-fold mixing implies 2-fold mixing by letting A=X. However, whether 2-fold mixing implies 3-fold mixing is surprisingly unknown.

Example: A common example of a dynamical system is a shift map. In this system, the space is a sequence space, and the transformation function shifts a sequence one term to the left or right. For example, X = \{0,1\}^{\mathbb Z}, f(x)_i := x_{i+1}, with \mu being the coin-toss measure. This system is mixing since different elements in any sequence are independently chosen, so any "cylinder set", which is a set defined by restricting a finite number of elements of the sequence, becomes independent from its translation eventually.
